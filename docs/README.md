# Brainworm Documentation

Brainworm is an xAI-powered workspace for chat, coding, research, image work,
and voice playback. It is a tool for creating and learning, not a person or a
relationship.

## Safety and use policies

- [AI Output Disclaimer and Conditions of Use](ai-output-disclaimer.md)
- [Not a Companion](not-a-companion.md)

Please read both documents before using or deploying Brainworm. In particular:

- Brainworm is not intended for children or child-directed use.
- Brainworm is not a friend, companion, therapist, or romantic partner.
- AI output can be wrong, unsafe, biased, or inappropriate. Verify it before
  acting on it.
- Do not rely on Brainworm for emergencies, crisis response, professional
  advice, or other safety-critical decisions.
